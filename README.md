# SQL Retail Sales Analysis

## Project Overview
This project analyzes retail sales data using PostgreSQL. The analysis includes database creation, data cleaning, exploratory analysis, and business-driven SQL queries.

## Objectives
- Clean and validate sales data
- Explore customer and product behavior
- Analyze sales performance
- Generate business insights

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- Window Functions
- Common Table Expressions (CTE)
- CASE WHEN
- EXTRACT
- TO_CHAR

## Business Questions Solved
- Daily sales analysis
- Category-based performance
- Customer purchasing behavior
- Top customers by sales
- Monthly best-selling periods
- Sales shifts analysis

- `sql_query_p1.sql` → Main SQL project file
